# COLOMBIANA MYA

## SETUP

### BACKEND

1. install dependencies

    ```bash
    npm/pnpm/yarn install
    ```

2. add env variables (Checkout .env.template)

3. setup db

    ```bash
    docker-compose up -d
    ```

4. start dev server

    ```bash
    pnpm/npm/yarn start:dev
    ```

5. seed db

    ```bash
    http://localhost:3000/api/seed
    ```
